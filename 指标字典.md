# 指标字典

> 本文档基于 `final.xlsx` 中的「指标统计」Sheet，定义系统所有核心字段的归属、来源、计算逻辑和权限。
> 开发过程中，所有核心报表指标必须从本字典中追溯来源，不得仅根据页面展示字段直接实现。

---

## 字段定义规范

每个字段需明确以下属性：

| 属性 | 说明 |
|------|------|
| 一级分类 | 对应系统一级菜单或业务域 |
| 所属表 | 对应系统逻辑表（style/sku/promotion/settlement等） |
| 中文列名 | 业务名称 |
| 英文字段名 | 数据库字段名 |
| 字段类型 | 人工录入 / 外部导入 / 系统计算 / 配置项 |
| 数据来源 | 千牛 / 万相台 / 灰豚 / 站外推广 / 商品成本表 / 人工等 |
| 数据库类型 | VARCHAR / DECIMAL(12,2) / DATE / BOOLEAN / UUID / JSONB 等 |
| 计算逻辑 | 系统计算字段的公式或伪代码 |
| 是否必填 | 创建/审核/发布/结算等不同状态下是否必填 |
| 是否允许人工覆盖 | 外部导入或系统计算字段是否允许人工修改 |
| 字段权限 | 普通可见 / 敏感字段 / 财务字段 / 管理员字段 |
| 参与报表 | 是否参与工作进度/发文进度/投产报表等统计 |
| 校验规则 | 类型/范围/枚举/唯一性/关联检查 |
